# ACC102 Assignment - Social Media Engagement Analysis

## Project Overview
This project analyzes a social media engagement dataset to understand how different factors influence user engagement.

The dataset includes posts from platforms like Facebook, Instagram, and Twitter, along with engagement metrics such as likes, comments, and shares.

The findings can help improve content strategies for social media marketing.

## Dataset Description
The dataset used in this project is sourced from Kaggle: *Social Media Engagement*.

Key features of the dataset include:
- Multiple social media platforms (Facebook, Instagram, Twitter)
- Different post types (image, video, carousel, poll, etc.)
- Engagement metrics (likes, comments, shares)
- Post timestamp for time-based analysis

This dataset is suitable for:
- Trend analysis
- Data visualization
- Understanding user engagement behavior

## Project Structure
- `analysis.ipynb` — Jupyter Notebook containing data analysis and visualization
- `social_media_engagement1.csv` — Dataset file
- `requirements.txt` — Required Python libraries
- `README.md` — Project documentation

## Analysis Tasks
In this project, the following analyses were performed:
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Comparison of engagement across platforms
- Analysis of post types and their performance
- Visualization of engagement trends

## Key Insights
Some key findings from the analysis include:
- Engagement varies significantly across different platforms
- Certain post types (e.g., video or carousel) tend to perform better
- Posting time may influence engagement levels
- User interaction patterns differ between platforms
